let和const

let声明变量只在局部有效

let具有没有变量提升 不允许重复声明 块级作用域的特性。

const声明一个不可改变的常量。

const不可改变的是指针的地址 地址内如果发生改变不可探测。

ES6共有6种声明变量的方法

  1. let
  2. const
  3. var
  4. function
  5. import
  6. class
    (ES5只有前两种)
