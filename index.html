实例生命周期

Vue2.0 Description
beforeCreate 组件实例刚被创建，组件属性计算之前，如el属性等
created 组件实例创建完成，属性已绑定，DOM未生成
beforeMount 模板编译/挂载之前
mounted 模板编译/挂载之后
beforeUpdate 组件更新之前
updated 组件更新之后
beforeDestory 组件销毁前
destoryed 组件销毁后
